### 👋 &nbsp;About

I'm a software developer who likes turning rough ideas into polished, dependable tools — from **local-first desktop apps** to **classical machine learning** and the **services** that sit behind them. I care about privacy, clean UX, and shipping things that actually get used.

- 🖥️ &nbsp;**Public work** — [`Live_transcribe`](https://github.com/lookynch/Live_transcribe): a local-first Windows dictation app that runs **Whisper on-device** for private, offline transcription, with optional **OpenAI** text refinement, push-to-talk, a floating overlay, and seamless **Velopack** auto-updates. Plus a hands-on [`scikit-learn`](https://github.com/lookynch/Machine-Learning) ML playground.
- 🔒 &nbsp;**Behind the scenes** — most of my day-to-day lives in private repositories: **internal tools & automation**, **line-of-business desktop and web apps**, **data / ML & AI** integrations, and the **web APIs & backends** that tie everything together.
- 🎯 &nbsp;**How I build** — local-first where it counts, pragmatic everywhere else, and always chasing that last 5% of polish.
